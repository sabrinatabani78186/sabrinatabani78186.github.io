⚙️ Static Site Generators

Software that generates static HTML pages from text files

  • No database needed
  • Fast and secure
  • Perfect for blogs & docs

Popular SSGs

  • 🔷 Jekyll — Used in GitHub Pages
  • 🔶 Hugo — Written in Go, very fast
  • ⚛ Next.js — JavaScript & React
  • 🌟 Eleventy — Supports 10 template languages

🐙 GitHub Pages

Free hosting service by GitHub for static websites

  • Free to use
  • Powered by Jekyll
  • Uses your GitHub repository
  • Automatic deployment

How It Works

  1. Create a repository
  2. Add Markdown files
  3. Enable GitHub Pages
  4. Your site is live! 🚀

URL format: username.github.io
