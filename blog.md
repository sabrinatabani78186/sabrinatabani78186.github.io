# My Blog

## Post 1: What is Markdown?
Markdown is a lightweight markup language that 
allows you to write formatted text using plain text.

### Why I like Markdown:
- It is simple to learn
- It is readable as plain text
- It converts easily to HTML

## Post 2: What is GitHub Pages?
GitHub Pages is a free hosting service that 
takes files from a GitHub repository and 
publishes them as a website.

### Benefits of GitHub Pages:
- Free hosting
- Easy to use
- Works with Jekyll
- No server needed

## Post 3: What is a Static Site Generator?
A static site generator (SSG) creates HTML pages 
from plain text files before they are served.

### Examples of SSGs:
- Jekyll
- Hugo
- Next.js
- Eleventy
